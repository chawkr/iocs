# IOCs

Indicators of Compromise (IOCs) for public sharing.

## Overview

This repository contains IP addresses and other indicators automatically retrieved based on threat hunting profiles created by Chawkr CTI team, published on https://chawkr.com/threat-intel.

## File Format

Each file in this repository corresponds to a security profile and contains:
- Unique IP addresses, one per line
- Files are overwritten on each update, empty files indicate no results found
